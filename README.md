# España vs Argentina - Final Mundial 2026

Landing page para promocionar la gran final de la Copa Mundial de la FIFA 2026 entre las selecciones de **España** y **Argentina**.

## Sobre el proyecto

Este repositorio contiene el código fuente de una página web diseñada para generar expectativa y difundir información sobre el partido final del Mundial 2026, incluyendo:

- Fecha, hora y sede del encuentro
- Presentación de ambas selecciones
- Contenido visual y promocional del evento
- Información de interés para los aficionados
